IXIOXI

Description

  • github.ixioxi.com

It is about the project of a company in which it is question of the implementation of a system OAuth 2.0 which will allow the deployment of several SAAS applications on a main platform with the aim of facilitating the connection to all the SAAS applications with only one account which will be created on the main platform in order to avoid the creation of an account on each application. In a second step, we have the implementation of the database multi-tenancy system in each SAAS application to allow the separation and the securing of the data of each company using the applications of the platform.
